# 音画同步

ClipFlow 提供专业的音画同步功能。

## 自动同步

系统自动分析音频波形，实现精准同步：

1. **静音检测** - 识别无声段落
2. **语音对齐** - 文字与语音匹配
3. **节奏检测** - 自动调整片段时长

## 手动调整

### 时间轴微调

拖动片段边缘调整起始时间：

- **左侧拖动**: 调整入点
- **右侧拖动**: 调整出点

### 偏移校正

使用「偏移」功能进行精确调整：

```
正偏移: 音频提前
负偏移: 音频延后
```

## 同步设置

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 静音阈值 | 检测静音的音量 | -40dB |
| 最小静音时长 | 最短静音段落 | 0.5s |
| 场景阈值 | 场景切换敏感度 | 0.3 |

## 常见问题

**音画不同步怎么办？**

1. 检查原始视频是否本身不同步
2. 尝试调整偏移值
3. 重新分析视频
