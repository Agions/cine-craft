# 解说混剪

ClipFlow 支持一键生成解说视频和混剪视频。

## 混剪模式

支持从多个视频中智能选取片段，生成精彩混剪：

### 配置选项

- **片段数量**: 最多选择 10 个片段
- **排序方式**: 随机 / 场景类型 / 情感分析 / 运动强度
- **转场效果**: 淡入淡出 / 溶解 / 切换 / 滑入

### 支持的风格

| 风格 | 说明 |
|------|------|
| narrative | 叙事风格 |
| humor | 幽默风趣 |
| professional | 专业正式 |
| casual | 轻松随意 |

## 输出设置
