## 简介

ClipFlow 是一款专业的 AI 视频内容创作平台，帮助创作者快速完成视频创作。

### 为什么选择 ClipFlow？

| 特性 | 传统方式 | ClipFlow |
|------|----------|----------|
| 字幕提取 | 手动听写 | AI 自动识别 |
| 文案撰写 | 逐字撰写 | AI 智能生成 |
| 配音合成 | 录音棚录制 | TTS 一键生成 |
| 剪辑效率 | 数小时 | 几分钟 |

### 支持的视频比例

- 📱 9:16 - 抖音、快手
- 📺 16:9 - 西瓜视频、B站
- 📷 1:1 - 小红书
- 📺 4:3 - 传统比例
