# 智能剪辑

ClipFlow 提供强大的智能剪辑功能，让视频编辑变得简单高效。

## 一键智能剪辑

点击工具栏中的 🤖 按钮，ClipFlow 将自动：

1. **视频分析** - 识别场景、检测镜头
2. **场景检测** - 智能分割视频场景
3. **静音检测** - 自动识别静音段落
4. **智能剪辑** - 生成最优剪辑方案

## 手动剪辑

### 添加片段

点击「添加片段」按钮，在当前时间点插入新片段。

### 分割片段

选中片段后，点击分割按钮可将片段一分为二。

### 删除片段

选中片段后，点击删除按钮移除该片段。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Space | 播放/暂停 |
| ← | 后退 5 秒 |
| → | 前进 5 秒 |
| Ctrl+S | 保存项目 |
| Ctrl+Z | 撤销 |
| Ctrl+Y | 重做 |
