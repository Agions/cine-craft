# 快速开始

5 分钟快速上手 ClipFlow！

## 第一步：创建项目

1. 打开 ClipFlow 应用
2. 点击「新建项目」
3. 输入项目名称
4. 选择目标视频比例
5. 点击「创建」

## 第二步：上传视频

1. 点击「上传视频」按钮
2. 选择本地视频文件
3. 等待上传完成
4. 视频将自动进行分析

## 第三步：AI 分析

系统会自动：
- 提取文字内容 (OCR)
- 识别语音并生成字幕 (ASR)
- 检测场景变化
- 分析关键帧

## 第四步：选择功能

三大核心功能任你选择：

### 🎬 AI 视频解说
适合：教程、评测、产品介绍
- AI 自动生成解说文案
- 多种配音音色可选
- 音画自动同步

### 👤 AI 第一人称解说
适合：Vlog、个人分享
- 以第一人称视角叙述
- 更贴近观众
- 情感更丰富

### ✂️ AI 混剪
适合：精彩集锦、混剪视频
- 自动识别高光片段
- 添加节奏感强的背景音乐
- 自动生成旁白

## 第五步：预览和调整

- 可以预览生成结果
- 支持手动调整文案
- 可以更换配音音色
- 可以调整语速

## 第六步：导出

1. 选择导出格式
2. 设置视频质量
3. 点击「导出」
4. 等待渲染完成
5. 下载成品视频

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl + S | 保存项目 |
| Ctrl + Z | 撤销 |
| Ctrl + Y | 重做 |
| Space | 播放/暂停预览 |
