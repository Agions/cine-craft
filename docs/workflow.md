# 工作流程

ClipFlow 提供完整的 AI 视频创作流程。

## 流程概览

```
┌────────────┐    ┌────────────┐    ┌────────────┐    ┌────────────┐    ┌────────────┐    ┌────────────┐
│  创建项目   │ -> │  上传视频   │ -> │  AI 分析   │ -> │ 生成文案   │ -> │ 视频合成   │ -> │  导出视频  │
└────────────┘    └────────────┘    └────────────┘    └────────────┘    └────────────┘    └────────────┘
   project        video          analysis         script        synthesis         export
```

---

## 步骤详解

### 1️⃣ 创建项目

**目的**：建立视频创作的基本单元

**操作**：
- 输入项目名称
- 选择目标视频比例
- 选择使用的核心功能

**产出**：项目 ID 和基础配置

---

### 2️⃣ 上传视频

**目的**：获取要处理的视频素材

**操作**：
- 点击上传按钮
- 选择本地视频文件
- 支持拖拽上传
- 支持批量上传

**产出**：视频文件信息和预览图

---

### 3️⃣ AI 分析

**目的**：理解视频内容

**自动完成**：
- OCR 文字识别
- 语音转字幕 (ASR)
- 场景检测
- 关键帧提取
- 物体识别

**产出**：视频分析报告，包括：
- 文字内容
- 字幕数据
- 场景列表
- 关键帧

---

### 4️⃣ 生成文案

**目的**：创作视频解说内容

**选择功能**：
- 🎬 AI 视频解说
- 👤 AI 第一人称解说
- ✂️ AI 混剪

**操作**：
- 选择文案风格
- 调整内容要点
- 预览文案效果

**产出**：完整的视频文案脚本

---

### 5️⃣ 视频合成

**目的**：将文案转化为成品视频

**处理内容**：
- 语音合成 (TTS)
- 音画同步对齐
- 添加背景音乐
- 应用视频特效
- 画面裁剪调整

**产出**：合成后的视频预览

---

### 6️⃣ 导出视频

**目的**：输出最终成品

**设置选项**：
- 视频格式 (MP4/MOV)
- 分辨率 (720p/1080p/4K)
- 画质 (普通/高/最高)
- 是否烧录字幕

**产出**：可分享的视频文件

---

## 数据流转

| 步骤 | 输入 | 处理 | 输出 |
|------|------|------|------|
| 创建项目 | 用户输入 | 保存元数据 | project |
| 上传视频 | 视频文件 | 解析视频信息 | video metadata |
| AI 分析 | video | OCR/ASR/场景检测 | analysis + subtitles |
| 生成文案 | analysis | LLM 生成 | script |
| 视频合成 | video + script | TTS + 音画同步 | final video |
| 导出视频 | final video | 编码渲染 | export file |

---

## 注意事项

- 📌 每一步都支持预览和调整
- 💡 可以随时返回上一步修改
- ⚡ 进度实时显示
- 🔄 支持断点续传
